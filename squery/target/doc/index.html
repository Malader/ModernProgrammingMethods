Squery 0.1.0-SNAPSHOT

Released under the EPL-2.0

Навигация, модификация и валидация данных в виде S-выражений (аналог XPath/XML Schema/XSLT для Clojure).

Installation

To install, add the following dependency to your project or build file:

[squery "0.1.0-SNAPSHOT"]
